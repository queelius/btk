# Installation

## Requirements

- Python 3.8 or higher
- pip (Python package installer)

## Install from PyPI

```bash
pip install bookmark-tk
```

## Install from Source

```bash
git clone https://github.com/queelius/bookmark-tk.git
cd bookmark-tk
pip install -e .
```

## Verify Installation

```bash
btk --version
```

## Optional Dependencies

### For PNG Graph Export

```bash
pip install cairosvg
```

### For Development

```bash
pip install -e ".[dev]"
```
